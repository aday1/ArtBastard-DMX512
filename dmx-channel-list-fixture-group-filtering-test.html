üß™ Testing Procedure

Step 1: Test Active Fixtures Filter

  1. Start the application and navigate to the DMX Control Panel
  2. Set some fixtures to non-zero DMX values (use SuperControl or individual sliders)
  3. In the filter dropdown, select "Active Fixtures Only"
  4. Expected: Only channels from active fixtures should be displayed
  5. Change DMX values and verify the filter updates dynamically

Step 2: Test Group Filtering

  1. Create fixture groups in the application (if not already created)
  2. Open the DMX Control Panel filter dropdown
  3. Verify that "Group: [Name]" options appear for each group
  4. Select a group filter
  5. Expected: Only channels from fixtures in that group should be displayed
  6. Test with different groups to verify correct channel mapping

Step 3: Test Filter Integration

  1. Switch between different filter types
  2. Verify that pagination updates correctly for each filter
  3. Check that the status bar shows the current filter name
  4. Confirm that channel counts are accurate for each filter
  5. Test empty state messages when filters have no results

Step 4: Test Backwards Compatibility

  1. Verify that the "Selected Only" touch mode button still works
  2. Test existing channel range filters
  3. Ensure all existing functionality remains intact
  4. Check that external monitor touch controls still function
