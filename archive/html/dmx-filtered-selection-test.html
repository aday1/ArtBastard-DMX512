üß™ Testing the New Filters

1. Test Active Fixtures Filter

  1. Open the application and navigate to Touch DMX Control Panel
  2. Set some DMX channels to non-zero values (create some lighting)
  3. In the filter dropdown, select Active Fixtures Only
  4. Expected: Only channels from fixtures with active values should be displayed
  5. Change some DMX values to zero and observe the filter update dynamically

2. Test Group-Based Filtering

  1. Go to Fixture Setup and create some fixture groups (e.g., "Stage Wash", "Moving Heads")
  2. Return to Touch DMX Control Panel
  3. Open the filter dropdown - you should see Group: [Group Name] options
  4. Select a group filter
  5. Expected: Only channels from fixtures in that group should be displayed
  6. Test pagination with the filtered group channels

3. Test Filter Integration

  1. Switch between different filter types (Selected, Active, Groups, Ranges)
  2. Verify pagination resets correctly when changing filters
  3. Test channel selection within filtered views
  4. Confirm that existing filters still work as expected
  5. Expected: Smooth transitions between all filter modes
