üß™ Test Procedures

1. TouchOSC Button Test

Step 1: Open the SuperControl component in your application
Step 2: Look for the blue "TouchOSC" button in the header area (top right)
Step 3: Click the TouchOSC button
Expected: The TouchOSC Export modal should open with export options and settings

2. OSC Help Button Test

Step 1: Locate the yellow "OSC Help" button next to the TouchOSC button
Step 2: Click the OSC Help button
Expected: The OSC Help modal should open showing OSC address reference

3. Button Styling Test

Step 1: Verify both buttons have proper styling:
  • TouchOSC button: Blue color scheme with smartphone icon
  • OSC Help button: Yellow/amber color scheme with help circle icon
Step 2: Test hover effects - buttons should glow and lift slightly on hover
Expected: Both buttons should have modern, glowing appearance with smooth hover animations

4. OSC Help Modal Content Test

Step 1: Open the OSC Help modal
Step 2: Verify the modal contains these sections:
  • Basic Controls (/dimmer, /red, /green, /blue, /white)
  • Movement Controls (/pan, /tilt, /xy)
  • GOBO & Effects (/gobo, /gobo_rotation, /strobe, /focus)
  • Scene Control (/scene/save, /scene/prev, /scene/next)
  • Usage Notes section
Expected: All OSC addresses should be displayed with proper formatting and descriptions

5. Modal Functionality Test

Step 1: Test closing the OSC Help modal using the X button
Step 2: Test closing with the "Close" button at the bottom
Step 3: Test closing by clicking outside the modal (if implemented)
Expected: Modal should close properly with all methods

üéØ OSC Address Reference (From Modal)

The OSC Help modal should display these addresses:

Basic Controls

/dimmer - Master Dimmer (0.0 - 1.0) /red - Red Color (0.0 - 1.0) /green - Green Color (0.0 - 1.0) /blue - Blue Color (0.0 - 1.0) /white - White Color (0.0 - 1.0)

Movement Controls

/pan - Pan Position (0.0 - 1.0) /tilt - Tilt Position (0.0 - 1.0) /xy - Combined Pan/Tilt (x, y values 0.0 - 1.0)

GOBO & Effects

/gobo - GOBO Selection (0.0 - 1.0) /gobo_rotation - GOBO Rotation (0.0 - 1.0) /strobe - Strobe Speed (0.0 - 1.0) /focus - Focus/Zoom (0.0 - 1.0)

Scene Control

/scene/save - Save Current Scene (bang/trigger) /scene/prev - Previous Scene (bang/trigger) /scene/next - Next Scene (bang/trigger)
