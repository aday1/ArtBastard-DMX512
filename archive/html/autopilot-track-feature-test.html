Test 1: Basic Enable/Disable

  1. Locate the "Enable Autopilot Track" checkbox
  2. Click the checkbox to enable autopilot mode
  3. Verify the autopilot controls become visible
  4. Uncheck the box to ensure controls hide properly
  5. Re-enable for further testing

Test 3: Position Control

  1. Set track type to "Circle"
  2. Slowly move the "Track Position" slider from 0% to 100%
  3. Verify fixtures move smoothly along the circular path
  4. Note that position updates happen in real-time as you drag
  5. Test with other track patterns

Test 4: Size and Center Controls

  1. Set track size to different values (25%, 50%, 75%, 100%)
  2. Observe how the movement radius changes
  3. Adjust "Center Pan" and "Center Tilt" sliders
  4. Verify the track center moves to the new position
  5. Click "Center Track" button to reset to center (127, 127)

Test 5: Multi-Fixture Testing

  1. Select multiple fixtures or a group
  2. Enable autopilot and test movement
  3. Verify all selected fixtures move in sync
  4. Test with different selection modes (channels, fixtures, groups)
