# ArtBastard Installation

## Requirements

- Linux/macOS/Windows
- Node.js 20+
- npm 10+

## Install

From repository root:

- npm ci
- npm --prefix react-app ci

## Run

Linux/macOS:

- ./start.sh

Windows:

- .\start.ps1

App URL:

- http://localhost:3030

## Optional reset flags

- ./start.sh --reset
- .\start.ps1 -Reset

## Verify build and runtime

- npm run build
- npm run test:api-contract
- npm run test:touchosc-workflow
