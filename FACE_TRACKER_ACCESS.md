# How to Access Face Tracker in the UI

## Quick Access Steps

1. **Go to the Dashboard** (Main page)
   - Click "Dashboard" in the left navigation menu (or it should be the default page)

2. **Find the Component Toolbar**
   - Look for a toolbar panel on the **right side** of the screen
   - It says "Component Toolbar" at the top
   - If you don't see it, it might be minimized (look for a small toolbox icon button)

3. **Expand the Toolbar** (if collapsed)
   - Click the chevron button (up/down arrow) to expand
   - Or click the restore button if it's minimized

4. **Select DMX Category**
   - In the toolbar, find the "Category" dropdown
   - Select "DMX" (or "All Components")

5. **Find Face Tracker**
   - Scroll through the component list
   - Look for "Face Tracker" with a user-circle icon
   - It should be near the top (after DMX Control Panel)

6. **Add to Panel**
   - **Drag** the "Face Tracker" component from the toolbar
   - **Drop** it into any panel (Left Panel, Right Panel, or Bottom Panel)
   - The component will appear and you can start using it!

## Alternative: Quick Add Button

If the toolbar is hard to find, you can also:
- Look for a small toolbox icon (🔧) in the top-right area
- Click it to expand the Component Toolbar
- The toolbar can be docked, minimized, or floating

## Troubleshooting

**Can't see the toolbar?**
- Make sure you're on the Dashboard/Main page (not Settings, Fixtures, etc.)
- Check if it's minimized - look for a small circular button with a toolbox icon
- Try refreshing the page

**Face Tracker not in the list?**
- Make sure "DMX" category is selected (or "All Components")
- Check that the component registry loaded properly (check browser console)

**Component won't drag?**
- Make sure you're clicking and holding on the component card
- Drag it to a panel area (the panels have drop zones)

## Quick Start

Once you've added the Face Tracker component:
1. Click the **"Start"** button in the component
2. Grant camera permissions when prompted
3. Position yourself in front of the camera
4. The moving head will follow your face!
