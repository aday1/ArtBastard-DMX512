üéØ Timeline Controls Usage

Basic Timeline Control:
  1. Select a timeline sequence from the dropdown
  2. Click the Play/Stop button to control playback
  3. Use the progress bar to monitor playback position
Loop & Bounce Modes:
  1. Loop Button: Enable continuous repeat of timeline
  2. Bounce Button: Enable ping-pong playback (forward then reverse)
  3. Reverse Button: Play timeline in reverse direction
  4. All modes can be combined for complex playback patterns
Speed Control:
  1. Use the speed slider for precise control (0.1x - 5.0x)
  2. Click preset buttons for common speeds: 0.5x, 1.0x, 1.5x, 2.0x
  3. Speed affects all timeline playback modes

üéπ MIDI Timeline Trigger Setup

Setting Up MIDI Triggers:
  1. Select the timeline sequence you want to control
  2. Click "Learn MIDI Trigger" button
  3. Press the desired MIDI key/button on your controller
  4. The mapping will be automatically created and displayed
  5. Test by pressing the MIDI key to start/stop the timeline
MIDI Mapping Features:
  • Toggle Behavior: First press starts timeline, second press stops it
  • Multiple Mappings: Assign different MIDI keys to different timelines
  • Clear Mappings: Remove individual or all MIDI assignments
  • Channel Support: Works with all 16 MIDI channels
